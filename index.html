Player 1's Turn
Moves Left: 64
Player 1 Points: 0
Player 2 Points: 0

Objective

The objective of the game is to have the highest score after 64 turns by strategically placing 3x3 blocks of different colors on an 8x8 grid.

Setup

The game is played on an 8x8 grid.
Each player starts with a score of 0.
There are four colors available: red, yellow, green, and blue.

Gameplay

Turns: Players take turns placing a 3x3 block on the grid. Each turn, a player can place a block of any of the four colors (red, yellow, green, or blue).

Placement: A 3x3 block can be placed anywhere on the grid, even partially outside the boundaries. Only the parts within the grid count.

Color Replacement and Scoring:

When a block is placed over existing colored squares, points are scored based on the color of the squares being covered and the color of the new block.

Points are awarded as follows when replacing colors:

Red to Yellow: +1 point
Red to Green: +2 points
Red to Blue: +3 points
Red to Red: 0 points
Yellow to Red: +3 point
Yellow to Green: +1 point
Yellow to Blue: +2 points
Yellow to Yellow: 0 points
Green to Red: +2 points
Green to Yellow: +3 point
Green to Blue: +1 point
Green to Green: 0 points
Blue to Red: +1 points
Blue to Yellow: +2 points
Blue to Green: +3 point
Blue to Blue: 0 points

Ending the Game

The game ends after 64 turns (each player has 32 turns).

The player with the highest score at the end of 64 turns wins the game.

Additional Rules

No Adjacent Placement Restriction: There is no restriction on placing a block adjacent to or overlapping a previously placed block.

Valid Placement: A block can be placed partially outside the grid, but only the parts within the grid count for scoring.

Turn Indicator: An indicator should be used to show which player's turn it is.

Score Display: Both players' scores should be displayed and updated in real-time as the game progresses.

Game End Announcement: At the end of the game, the winner should be announced based on the final scores.
